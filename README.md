# 🌡️ Smart Temperature Fan Control System

## 📌 Tujuan Project

### 1. Monitoring Suhu Otomatis
Membangun sistem monitoring suhu menggunakan sensor **TMP36** yang mampu membaca suhu ruangan secara **real-time**, kemudian menampilkan data suhu pada **LCD 16x2** dan **Serial Monitor**.

### 2. Kontrol Kecepatan Kipas Berdasarkan Suhu
Mengimplementasikan logika percabangan **if, else if, else** serta metode **PWM (Pulse Width Modulation)** pada Arduino untuk mengatur kecepatan kipas secara otomatis sesuai rentang suhu yang terdeteksi.

### 3. Implementasi Fitur Hardware Arduino
Menerapkan berbagai fitur pada **Arduino Uno**, seperti:
- **ADC** untuk membaca sinyal analog sensor  
- **I2C** untuk komunikasi dengan LCD  
- **UART** untuk debugging melalui Serial Monitor  
- **GPIO** untuk mengontrol input dan output sistem
